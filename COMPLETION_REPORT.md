## 🚀 Команды для запуска

```bash
# Компиляция
mvnw.cmd clean compile

# Сборка
mvnw.cmd package -DskipTests

# Запуск через Spring Boot
mvnw.cmd spring-boot:run

# Запуск через IDE
# Run → Run 'RestaurantApplication'

# Развертывание в Tomcat
# 1. copy target\restaurant-1.0-SNAPSHOT.war C:\apache-tomcat-10.1.50\webapps\
# 2. net stop Tomcat10
# 3. net start Tomcat10
```
